## Installation

**Python 3**.

Required libraries:

- qiskit
- numpy
- pandas
- matplotlib
- asyncio

---

## Project Motivation

This project is part of a practical exploration of Quantum Machine Learning concepts through implementation.

Model specifications:

- **Input:** Binary features
- **Output:** Probability prediction (0–1)
- **Target error:** ≤ 0.05

---

## File Descriptions

- **QML01_project_code.ipynb** – implementation of the QML model, including data preprocessing, parameterized quantum circuit construction, training, and evaluation.
